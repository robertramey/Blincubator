Pending Website Feature Enhancements and Bug Fixes

This needs some more feature enhancement and bug fixes. If you have interested in helping out please contact me. Expecially useful would be persons with experience with Wordpress. Here is a list of current issues which need help/improvement.

Permalinks

Given a permalink with the name http://www.blincubator.com/bi_library/safe-numerics I expect to be able to jump directly to the library page for the safe-numerics library. I would like to be able to fix this but so far I have been unable to do so.

Library Statistics

Right now we you SlimStat to gather and monitor statistics on each library as well as the web site as a whole. It's a great package and has served us very well. Still there a few issues which I would like to see resolved.

  • users can only see the slimstat statistics pages if they are logged in to the site. I would like to eliminate this requirement but haven't been able to figure out how to do this.

  • On the page for each library I'd like to gather and display the following

    • number of times the repo link has been invoked

    • number of times the download link has been invoked

    • number of times the documentation link has been invoked

  • I'd like to get a list of all the users who downloaded and/or followed the link to the library repo. I want to do this so I can email them a reminder to review and/or comment on the library after they've had a chance to evaluate it.
