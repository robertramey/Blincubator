Boost Library Writers Workshop

Boost library Incubator is proud announce the Library Writers Workshop to be hosted as an Open Content session at CPP Con 2014 Monday, September 8 at 8:30 PM. The goal of this workshop is to help potential library authors acquire the tools, skills and motivation to develop a boost library candidate and submit it to the Boost Library Incubator. This is an opportunity for library authors to get feedback on their progress and encouragement for their efforts.

Note

One of the great things about this workshop is that you don't have to be registered for CPP Con to participate !!! You can just drop in and participate if you're in the area !!!. I'm hoping to see some interesting efforts. (Hmmm - what about remote participation?).

Requirements

There two types of attendees: Library Authors and Observers. The method used will be peer critique. In order to participate as an Observer, all one has to do is show up. In order to participate as an author, one must:

  • Have a library in progress.

  • Have documentation and perhaps code which is accessible online.

  • Create a comment entry below which includes

    • Name of the library

    • Short description of what the library does

    • Links to documentation and perhaps code

    Create your entry as soon as you know you'll be attending the workshop as an author. This way we'll be able to save you a space.

Program

The workshop will operate on the following schedule:

  • 8:30 PM - 8:35 PM I will introduce the workshop with a short summary of these rules and answer any questions.

  • 8:35 PM - 9:00 PM Each library author will make a very short presentation (~3 min) of what his library attempts to accomplish.

  • 9:00 PM Authors draw lots to select a library other than their own. Observers are also permitted to draw lots or just select a library they might be interested in. Observers might choose to leave at this point.

  • 9:00 PM - 10:00 PM Each selected library is “critiqued” by reading through the documentation and perhaps code while building a contemporaneous log on the experience. This log will be posted as a comment below.

  • 10:00 PM - Pencils down - wrap up. Authors may read and optionally reply to critiques at their leisure.

I will be available to answer questions and provide encouragement. Hopefully I can convince another boost library author to attend
